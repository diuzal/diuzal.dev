01 ✔

Modrinth App'i İndir (Yüklü Değilse)

Modpack'i yönetmek için kullandığımız başlatıcı olan Modrinth App'i indirip kur.

ℹ modrinth.com/app adresine git ve işletim sistemine uygun yükleyiciyi indir — Windows, Mac veya Linux. Çalıştır ve kurulumu tamamla.
02 ✔

Microsoft Hesabınla Giriş Yap

Modrinth App'i aç, sağ üst köşedeki Giriş Yap butonuna tıkla. Minecraft: Java Edition satın aldığın Microsoft hesabınla giriş yap.

⚠ Sunucuya girebilmek için gerçek bir Minecraft: Java Edition lisansına ihtiyacın var. Crack hesaplar bu sunucuda çalışmaz.
03 ✔

Modpack'i Yükle

Önce modpack dosyasını indir, ardından Modrinth App'e ekle:

Modpack'i İndir (.mrpack)

Dosyayı indirdikten sonra Modrinth App'e ekle:

  1. Sol kenar çubuğundaki + butonuna tıkla
  1. Açılan ekranda Modpack seçeneğine tıkla
  1. Import Modpack'e tıkla ve indirdiğin .mrpack dosyasını seç
04 ✔

Modpack'i Başlat

Instance üzerindeki Oyna butonuna tıkla. İlk açılışta tüm modlar indirilir — bağlantı hızına göre 1–2 dakika sürebilir.

✓ Ana menünün tamamen yüklenmesini bekle, ardından bir sonraki adıma geç.
05 ✔

Sunucuyu Ekle ve Katıl

Minecraft ana menüsünden:

  1. Çok Oyunculu'ya tıkla
  2. Sunucu Ekle'ye tıkla
  3. Adrese 100.122.30.65 gir
  4. Bitti'ye tıkla, ardından sunucuya çift tıkla
✓ Sunucu yeşil ping ikonu gösteriyorsa hazırsın — tıkla ve Sunucuya Katıl!
